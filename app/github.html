Estructura del Proyecto

Organización de carpetas y archivos utilizados en el proyecto. 

                        taller3/
                        ├── app/
                        |       ├── forms.html
                        |       ├── tables.html
                        |       └── github.html
                        ├── assets/
                        |       ├── css/
                        |       ├── img/
                        |       └── js/
                        ├── index.html
                        └──README.md

Explicación de la estructura del Proyecto

Elemento Tipo Descripción Ubicación
taller3/ Carpeta principal Contiene todos los archivos y recursos del proyecto. Raíz del proyecto
app/ Carpeta Almacena las páginas secundarias del sitio web. taller3/app
forms.html Archivo HTML Página que contiene los formularios de registro e inicio de sesión. app/forms.html
tables.html Archivo HTML Página donde se muestran las tablas del proyecto. app/tables.html
github.html Archivo HTML Página que explica el uso de Git y GitHub. app/github.html
assets/ Carpeta Contiene todos los recursos estáticos del proyecto. taller3/assets
css/ Carpeta Archivos de estilos utilizados para el diseño del sitio. assets/css
img/ Carpeta Almacena las imágenes utilizadas en la página. assets/img
js/ Carpeta Contiene los archivos JavaScript y Bootstrap necesarios para la funcionalidad del sitio. assets/js
index.html Archivo HTML Página principal del proyecto desde donde se accede a las demás secciones. Raíz del proyecto
README.md Archivo Markdown Documento con la descripción general del proyecto, instrucciones y documentación. Raíz del proyecto

Configuración de Git

Paso 1: Inicializar Git
git init
Paso 2: Configurar Usuario
git config --global user.name "Tu Nombre"

git config --global user.email "correo@gmail.com"
Paso 3: Verificar Configuración
git config --global --list

Creación de Repositorio en GitHub

  1. Ingresar a GitHub.
  2. Seleccionar New Repository.
  3. Asignar un nombre al repositorio.
  4. Crear el repositorio.
  5. Conectar el proyecto local.
git branch -M main

git remote add origin URL_REPOSITORIO

git push -u origin main
